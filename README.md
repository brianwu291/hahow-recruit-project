* **UI Library:** React.
* **State Management:** Redux, Redux-Thunk.
* **CSS Library:** styled-component.

* **Coding Style:** extends airbnb with eslint.
* **Bundler:** Webpack.
* **Unit test:** jest.
* **Version Control:** Git, Github.
